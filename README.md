# Odd baby
> A starter theme for Drupal theming.

## Install with Yeoman
```
$ yo odd:baby
```

## Usage
Install dependencies
```
$ npm install
```

Watch files:
```
// Watch .scss (scss-lint, sass) and .js (eslint, browserify)
$ gulp watch
```

## Requirements
* Gulp
* Sass 3.4
* Scss lint

## Includes
* Bourbon
* Neat
